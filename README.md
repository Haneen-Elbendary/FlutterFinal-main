# Flutter Diet & Recipe Management App

## Overview
This Flutter application is designed to assist users in managing their diet and discovering recipes tailored to their selected ingredients. The app emphasizes user-friendly functionality, secure authentication, and personalized recipe recommendations, making healthy eating easier and more accessible.

---

## Features

- **User Authentication:** Secure sign-up and login functionality.
- **Recipe Search:** Search for recipes by selecting specific ingredients.
- **Detailed Recipe Information:** View comprehensive details for each recipe.
- **Favorite Recipes:** Save and manage your favorite recipes for quick access.
- **User History:** Keep track of previously viewed recipes.

---

## Getting Started

### Prerequisites
Before you begin, ensure the following requirements are met:

1. **Flutter SDK**: Install Flutter by following the [Flutter Installation Guide](https://flutter.dev/docs/get-started/install).
2. **IDE**: Use a compatible IDE such as Visual Studio Code or Android Studio.
3. **Testing Environment**: Set up an emulator or connect a physical device for testing.

---

### Installation

1. **Clone the Repository:**
   ```bash
   git clone https://github.com/Haneen-Elbendary/FlutterFinal-main
   ```
2. **Navigate to the Project Directory:**
   ```bash
   cd FlutterFinal
   ```
3. **Fetch Dependencies:**
   ```bash
   flutter pub get
   ```
4. **Run the Application:**
   ```bash
   flutter run
   ```

### Backend API
The application integrates seamlessly with the backend API. For API setup and documentation, visit [FlutterFinalAPI Backend](https://github.com/Haneen-Elbendary/FinalAPI-main).
